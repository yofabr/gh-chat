# Contributing

Thanks for your interest in contributing to GH Chat! Please take a moment to read these guidelines before opening an issue or pull request.

## How to Contribute

1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies** for the area youâ€™re changing:
   - Backend: `cd backend && npm install`
   - Extension: `cd extension && npm install`
   - Frontend: `cd frontend && npm install`
3. **Make your changes** with clear commits.
4. **Add tests** or update existing ones when applicable.
5. **Ensure lint/build passes** for the part you touched.
6. **Open a pull request** with a concise description of the change and testing performed.

## Pull Request Checklist

- Descriptive title and summary of changes
- Tests added/updated (when applicable)
- No unrelated formatting changes
- Links to related issues (if any)

## Reporting Issues

- Search existing issues first to avoid duplicates.
- Include steps to reproduce, expected vs. actual behavior, and screenshots or logs when helpful.
